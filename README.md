# SmartBudget — Personal Finance Management System
## Project Overview
SmartBudget is a business analysis case study for a personal finance management system designed to help users track expenses, manage recurring payments, and improve budgeting habits.
The project demonstrates the full business analysis cycle:
- business problem identification,
- requirements gathering,
- process modeling,
- backlog management,
- SQL analysis,
- dashboard design.

---

## Business Problem
Many users struggle to:
- understand where their money goes,
- manage recurring subscriptions,
- control impulsive spending,
- maintain long-term budgeting habits.
The goal of SmartBudget is to simplify personal finance management through analytics and process automation.

---

## Project Goals
- Improve expense visibility
- Reduce missed recurring payments
- Increase user engagement with budgeting tools
- Provide actionable financial insights
