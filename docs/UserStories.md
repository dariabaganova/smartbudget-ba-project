# User Stories
## Epic: Expense Tracking
### US-001
As a user,  
I want to add my daily expenses,  
so that I can track where my money goes.

---

### US-002
As a user,  
I want expenses to be categorized automatically,  
so that I can better understand my spending habits.

---

## Epic: Budget Planning
### US-003
As a user,  
I want to set monthly spending limits,  
so that I can control my budget.

---

### US-004
As a user,  
I want to receive notifications when I exceed my budget,  
so that I can reduce unnecessary spending.

---

## Epic: Subscription Management
### US-005
As a user,  
I want to receive reminders about recurring payments,  
so that I can avoid missing subscriptions and bills.

---

## Epic: Analytics Dashboard
### US-006
As a user,  
I want to see monthly spending analytics,  
so that I can make better financial decisions.

---

### US-007
As a user,  
I want to visualize my expenses by category,  
so that I can identify my biggest spending areas.

---
